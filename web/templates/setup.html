IoT for LEDs

You want to connect your own Rapsberry Pi to the cloud?

Here you go!

  1. Install the packages python-requests and git.
  2. Clone the repository https://github.com/YtvwlD/iotled.git.
  3. Change into the subdirectory pi.
  4. Run "sudo make install".
  5. Connect the wires accordingly.
  6. Enjoy!
  7. When you're done, press the button to shut down.
